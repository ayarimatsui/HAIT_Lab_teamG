# HAIT_Lab_teamG
HAIT Lab Primary 東京3期 チームG 最終開発用リポジトリ

AIで栄養管理ができるwebアプリケーションをFlaskとYOLOで作りました。  
webページに料理の画像をアップロードすると、自動で何の料理が写っているのかが認識され、そのカロリーや栄養素が表示されます。
